#####################
## REGISTRY LOCAL: ##
#####################


OBS: - Para instalar APIC debemos contar con un registry local. Debe habilitarlo en el equipo cliente o en el master01, dependiendo desde donde ejecutará la instalación.
	 - El registry se debe habilitar en un equipo con Docker instalado.
	 - Deberá ejecutar este proceso y habilitar repositorios independientes en todos los clúster si la máquina con el registry no es vista por todas las redes.


1) Copie el archivo de imagen:

	Copie el siguiente archivo al equipo donde habilitará el registry local: apiconnect-image-tool-10.0.2.0.tar.gz
	
2) Ejecute el comando: 

	docker run -d -p 5000:5000 --restart=always --name registry registry:2
	
3) Ejecute el comando:
	
	docker login <IP-registry>:5000

4) Ejecute el comando:

	docker load < apiconnect-image-tool-10.0.2.0.tar.gz

5) Ejecute el comando:

	docker run --rm apiconnect-image-tool-10.0.2.0:latest upload 192.168.43.188:5000 --tls-verify=false

6) Verifique que las imágenes se han cargado con el siguiente comando:

	docker run --rm apiconnect-image-tool-10.0.2.0:latest version  --images 

7) En los master y worker agregue lo siguiente al archivo /etc/docker/daemon.json:

	{
	  "insecure-registries" : ["<IP-Registry>:5000"]
	}

	OBS: - <IP-Registry> debe ser reemplazado por la IP de la máquina donde se instala el registry local
		 - Si el archivo daemon no existe, creelo.

		 
	Luego, reinicie Docker con el comando: systemctl restart docker.service
